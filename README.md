## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/iretunde/Ping_Pong.git
   cd Ping_Pong

2. Run the game:

   ```bash
   python main.py

3. How to Play

**Player 1 Controls:**

- Move Up: `Q`
- Move Down: `A`

**Player 2 Controls:**

- Move Up: `Up Arrow`
- Move Down: `Down Arrow`
